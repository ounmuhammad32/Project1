What is Docker?

Docker The term Docker can refer to the docker project as a whole, which is a platform for developers and sysadmins to develop, ship, and run applications or the docker daemon process running on the host which manages images and containers (also called Docker Engine)

What is a docker container?

A container is a runtime instance of a docker image. a Docker container consists of, a Docker image, an execution environment, a standard set of instructions. The concept is borrowed from shipping containers, which define a standard to ship goods globally. Docker defines a standard to ship software.

Common Docker Commands

  • Compose - Compose is a tool for defining and running complex applications with Docker. With Compose, you define a multi-container application in a single file, then spin your application up in a single command which does everything that needs to be done to get it running. Also known as : docker compose up
  • Run – Runs a command in a new container. Also known as : docker start
  • Start – Starts one or more stopped containers. Also known as : docker stop
  • Stop – Stops one or more running containers. Also known as : docker build
  • Build – Builds an image form a Docker file. Also known as : docker build
  • Pull – Pulls an image or a repository from a registry. Also known as : docker pull
